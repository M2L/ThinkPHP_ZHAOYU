安装教程

short_open_tag = Off

1.上传程序并且解压到您的主机根目录；

2.还原数据库，这里采用的数据库备份是帝国备份王，如果用使用此工具备份数据呢？先访问http://您的域名/ebak进入帝国备份王，账号和密码分别是admin   123456


3.修改数据库配置文件

找到public/db_config.php文件，打开以后对照一下注释修改成自己的数据库信息

'DB_NAME'=>'zhaoyu3',                 ---------------------红色内容为你的数据库名称 
'DB_USER'=>'root',                      ---------------------红色内容为你的数据库用户名
'DB_PWD'=>'',                   ---------------------红色内容为你的数据库密码

4.重要的一步，删除缓存文件才可以登录后台

打开public\runtime把里面文件夹全部删除掉

5.访问http://您的域名/ohmyadmin.php就可以登录网站后台了，后台账号是admin密码是admin

把一些信息修改成自己的信息即可！
